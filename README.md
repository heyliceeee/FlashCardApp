# FlashCard App

Aplicação interativa para aprendizagem de vocabulário através de cartões digitais. Cada palavra é apresentada primeiro na língua original e, após alguns segundos, revela a tradução. O utilizador pode indicar se conhece ou não a palavra, permitindo que o conjunto de estudo seja atualizado automaticamente.

## ✨ Funcionalidades
- **Apresentação automática** do cartão e revelação da tradução após alguns segundos  
- **Botões de resposta** para indicar se a palavra foi aprendida  
- **Remoção automática** das palavras já conhecidas  
- **Gravação do progresso** num ficheiro dedicado  
- **Continuação do estudo** a partir do ponto onde ficou  
- **Mensagem final** quando todas as palavras forem aprendidas

## 📁 Estrutura dos dados
- Ficheiro principal com o conjunto inicial de palavras  
- Ficheiro gerado automaticamente com as palavras ainda por aprender  
- Ambos em formato CSV para fácil edição e expansão

## 🎯 Objetivo
Fornecer uma forma simples, rápida e eficaz de memorizar vocabulário através de repetição espaçada e interação direta.
